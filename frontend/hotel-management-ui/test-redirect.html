S3 Redirect Test Tool

Test Your S3 Bucket Redirection

This tool helps you verify if your S3 bucket redirection from the root URL to the /hotels path is working correctly.

Test Root URL Redirection

Expected Behavior:

When you click the button above, you should be automatically redirected to:

http://hotel-management-ui.s3-website.ap-south-1.amazonaws.com/hotels

Troubleshooting:

If the redirection doesn't work:

  • Check if you've properly configured the S3 bucket using one of the methods in the README
  • Verify that your S3 bucket has the correct permissions
  • Try clearing your browser cache or using an incognito/private window
  • Wait a few minutes for AWS to propagate the changes

How to Use This Test

  1. Click the "Test Root URL Redirection" button above
  2. A new tab will open with the root URL of your S3 website
  3. If the redirection is working correctly, you should be automatically redirected to the /hotels path
  4. Verify that the final URL in your browser's address bar ends with /hotels

For more detailed instructions on configuring S3 bucket redirection, refer to the S3-REDIRECTION-GUIDE.md file in this repository.
